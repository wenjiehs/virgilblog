## 🛠️ 技术栈

这个博客使用了以下技术栈：

- **VitePress** - 静态站点生成器
- **Vue 3** - 渐进式 JavaScript 框架
- **Vite** - 下一代前端构建工具
- **GitHub Pages** - 免费托管服务

## ✨ 特性

### 1. 现代化开发体验

VitePress 基于 Vite 构建，拥有极快的热重载速度：

```bash
# 启动开发服务器
npm run docs:dev

# 构建生产版本
npm run docs:build
```

### 2. Markdown 增强

支持丰富的 Markdown 语法扩展：

::: tip 提示
这是一个提示框
:::

::: warning 警告
这是一个警告框
:::

::: danger 危险
这是一个危险提示框
:::

### 3. 代码高亮

```javascript
// JavaScript 代码示例
function fibonacci(n) {
  if (n <= 1) return n
  return fibonacci(n - 1) + fibonacci(n - 2)
}

console.log(fibonacci(10)) // 55
```

```python
# Python 代码示例
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n - 1) + fibonacci(n - 2)

print(fibonacci(10))  # 55
```

- 📱 移动端优化

- 🖥️ 大屏适配

## 🚀 部署方式

本博客使用 GitHub Actions 自动部署到 GitHub Pages：

1. 推送代码到 `main` 分支

3. 部署到 GitHub Pages
